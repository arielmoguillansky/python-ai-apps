# AI Global Tip Calculator

A Python script that uses AI to determine the correct tip based on local customs and service quality.

## Features

- Culture-Aware: Adjusts tips for different countries (e.g., USA vs. Japan).

- Rating System: Factors in food, service, and ambiance.

- Bill Splitter: Calculates the cost per person.

## Setup

API Key: Create a .env file and add:

OPENAI_API_KEY=your_key_here

## How it Works

- Enter your bill total and country/city.

- Rate your experience (1-5 stars).

- The AI suggests a percentage based on international etiquette.

- The app calculates the final total and splits it by the number of people.
