# AI Band Name Generator

A creative Python tool that uses AI to generate unique band names based on music genre, hometown, and member count.

## Features

- Cultural Context: Uses the city's history and vibe to influence the name.

- Genre-Specific: Tailors the name to match the style of music.

- Unique Output: Ensures the generated name is distinct from the input data.

## Setup

API Key: Create a .env file and add:

OPENAI_API_KEY=your_key_here
